Welcome to bruker2bart's documentation!
=======================================

Bruker2bart is a Python package providing data format conversion between Bruker and `The Berkeley Advanced
Reconstruction Toolbox <https://mrirecon.github.io/bart/>`_ (BART) data formats.
It also provides several examples preforming reconstruction of Bruker data using BART.
